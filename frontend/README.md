# Frontend - Online Code Judge

This folder contains the React/Vite frontend for the Online Code Judge platform.

## Features

- Login and registration pages
- Problem browsing and filtering
- Code editor and submission workflow
- Console output and performance charts

## Requirements

- Node.js 18+

## Scripts

From the `frontend/` folder:

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint over the frontend source.

## Local development

Open a terminal in `frontend/` and run:

```bash
npm install
npm run dev
```

The frontend will run at `http://localhost:5173` by default.

## Notes

- No special frontend environment variables are required by default.
- Use `npm run dev` from the root to run both frontend and backend together.
